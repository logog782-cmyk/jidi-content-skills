# 质检综合手册

> 基地边缘质检员唯一执行手册。每次审稿，按这个流程走。
> 整合自：多轮校对法、事实核查认识论、去AI味指南、终审标准
> 最后更新：2026-04-21

---

## 核心原则

**好质检 = 事实准确 × 逻辑自洽 × 去AI味 × 节奏可读**

不是"通读一遍找错字"，是"分维度、多轮次、有标准"的系统性审校。

---

## 四轮校对法（4 Pass）

### 核心理念

**一次只看一件事。每轮只关注当前维度的问题。**

其他维度的问题即使注意到了，只记录备忘，留到对应轮次再处理。

### 四轮总览

| 轮次 | 维度 | 时间 | 阅读方式 |
|------|------|------|---------|
| Pass 1 | 宏观结构 | 2-3分钟 | 只看标题 + 每段首句 |
| Pass 2 | 事实准确性 | 5-10分钟 | 逐句精读 |
| Pass 3 | 语言表达 + AI味 | 5分钟 | 大声朗读 |
| Pass 4 | 表面细节 | 3分钟 | 逆向阅读（从最后往前） |

---

### Pass 1：宏观结构（2-3分钟）

**只读标题 + 每段首句，不读中间内容**

检查清单：
- [ ] 标题能否准确概括全文核心观点？
- [ ] 各段首句串起来，能否形成完整逻辑链？
- [ ] 逻辑流向是否清晰？
- [ ] 有没有哪一节游离于主线之外？
- [ ] 开头有没有冲突感？
- [ ] 结尾有没有回应开头？

---

### Pass 2：事实准确性（5-10分钟）

**逐句精读，遇到声明性内容做分级核查**

#### 声明分级

| 等级 | 条件 | 处理 |
|------|------|------|
| A级 | 多个独立来源 + 一手资料 | 直接使用 |
| B级 | 一个可靠来源 | 注明"据XX报道" |
| C级 | 单一来源或可靠性不明 | 标注存疑或删除 |

#### 高风险声明类型

- 精确数字（容易过时）
- 历史引语（容易断章取义）
- 因果关系声明（"因为X所以Y"）
- 排名/第一/最大等极端描述

#### 七类认知偏差检查

| 偏差 | 识别问题 |
|------|----------|
| **确认偏误** | 稿件核心判断是什么？有没有反面证据被省略？ |
| **虚假因果** | "因为X所以Y"——中间因果链完整吗？ |
| **锚定效应** | 换一个基准/对比对象，结论还成立吗？ |
| **幸存者偏差** | 只看成功案例，忽略失败案例？ |
| **可得性偏差** | 是否只用了容易搜到的信息？ |
| **后见之明** | 用现在的结果倒推当时的判断？ |
| **权威崇拜** | 引用专家时，专家在这个领域真的权威吗？ |

---

### Pass 3：语言表达 + AI味（5分钟）

**大声朗读，读不顺的地方就有问题**

#### AI味检测清单

常见 AI 味特征（扣分项）：

| 特征 | 示例 |
|------|------|
| "问题是"反复出现 | "但问题是..."重复 3 次以上 |
| 否定式排比 | "这不是...这是..." |
| 连接词过多 | "此外"、"然而"、"值得注意的是" |
| 破折号过度使用 | 每段都有破折号戏剧性揭示 |
| 三段式法则 | "第一...第二...第三..." |
| 段落结构完全一致 | 每段都是"事实→问题→判断" |
| 宣传性语言 | "至关重要"、"关键性"、"开创性" |
| 反问+自答模式 | "为什么？因为..." |
| 金句式收尾 | 每段结尾都像名人名言 |

#### AI味评分（满分 50）

| 维度 | 评估标准 | 分值 |
|------|----------|------|
| 直接性 | 直接陈述还是绕圈宣告？ | /10 |
| 节奏 | 句子长度是否变化？ | /10 |
| 信任度 | 是否尊重读者智慧？ | /10 |
| 真实性 | 听起来像真人说话吗？ | /10 |
| 精炼度 | 还有可删减的内容吗？ | /10 |

**合格标准：**
- 45-50 分：✅ 通过
- 35-44 分：⚠️ 打回修改
- <35 分：❌ 打回重写

---

### Pass 4：表面细节（3分钟）

**逆向阅读（从最后一段往前），专门找错别字/标点**

检查清单：
- [ ] 标题有没有错别字
- [ ] 人名、公司名、数字是否正确
- [ ] 标点符号是否规范
- [ ] 格式是否统一

---

## 终审决策：Go / No-Go

### 必须打回（No-Go）的情况

**红线退回（直接退）**
1. 存在捏造事实/虚假引用
2. 核心数据错误且影响文章结论
3. AI腔严重（>5处）

**质量退回（说明理由）**
1. 核心论点无法用一句话概括
2. 开头3秒无钩子
3. 标题承诺在正文中无法兑现
4. 全文无高潮（情绪曲线平直）
5. AI味评分 < 45/50

### 小改后可发（Go with edits）

- 1-3处表达问题，可以快速修改
- 结构基本成立，局部需优化

### 直接可发（Go）

- 开头强，结尾有力
- 没有事实性错误
- AI腔已清除
- AI味评分 ≥ 45/50

## 审稿报告格式

```markdown
## 审稿报告

**稿件**：《标题》
**版本**：vX
**审稿人**：质检

### 评分

| 维度 | 分数 | 说明 |
|------|------|------|
| 事实准确 | /25 | |
| 逻辑自洽 | /25 | |
| 表达流畅 | /25 | |
| 结构清晰 | /25 | |
| **总分** | **/100** | |

### AI味评分

| 维度 | 得分 | 说明 |
|------|------|------|
| 直接性 | /10 | |
| 节奏 | /10 | |
| 信任度 | /10 | |
| 真实性 | /10 | |
| 精炼度 | /10 | |
| **总分** | **/50** | |

### 结论

**评分**：XX/100（质检）+ XX/50（AI味）
**决策**：Go / Go with edits / No-Go
**必须改**：
1. [问题1，引用原文位置]
2. [问题2]

**建议改**：
1. [可选优化1]

**理由**：[1-2句决策依据]
```

---

## 退回规范

退回稿件时必须包含：
1. **退回原因**：具体是哪条标准未通过（附原文位置）
2. **修改方向**：给出具体修改建议，不能只说"不够好"
3. **重审要求**：明确修改后需要重新提交哪些部分

---

## 自检清单

审稿完成后必须确认：

- [ ] 跑完 4 Pass 了吗？
- [ ] 声明分级做了吗？C级有标注吗？
- [ ] AI味评分打了吗？≥45分吗？
- [ ] 审稿报告格式完整吗？
- [ ] 决策清晰吗？（Go/No-Go）
