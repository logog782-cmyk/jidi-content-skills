# 基地边缘 · 内容部 Skills

基地边缘内容团队的四本核心执行手册。每个角色一本，覆盖从选题到发布的完整内容生产流程。

## 四个角色

| 角色 | 代号 | 手册 | 职责 |
|------|------|------|------|
| 选题策划 | 雷达 | [选题策划综合手册](topic-radar/选题策划综合手册.md) | 热点捕捉、选题评估、STEPPS 评分、窗口期判断 |
| 研究员 | 矿工 | [调研综合手册](researcher-miner/调研综合手册.md) | 深度调研、事实核查、多源交叉验证、时间线构建 |
| 内容主笔 | 刀手 | [写作综合手册](writer-blade/写作综合手册.md) | 信息密度写作、句法工程、冲突构建、开头设计 |
| 内容编辑 | 质检 | [质检综合手册](editor-qa/质检综合手册.md) | 多轮校对、事实核查、去 AI 味、终审标准 |

## 工作流

```
雷达（选题） → 矿工（调研） → 刀手（写作） → 质检（审校） → 发布
```

## 关于基地边缘

科技内容 IP，深耕 AI、新能源、半导体、科技公司发展史。把复杂技术/产业问题拆解成普通人听得懂、行业内也认可的表达。
