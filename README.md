## 🎯 选题策划 · 雷达

**文件**：[选题策划综合手册](topic-radar/选题策划综合手册.md)

**核心价值**：从信息源监控到选题决策的完整 SOP。不靠"感觉选题"，靠量化标准和结构化流程。

**适用场景**：
- 建立系统化的选题机制
- 日常选题扫描和热点捕捉
- 选题评审和优先级排序
- 内容团队选题标准统一

**关键方法**：
- **STEPPS 评分体系**：社交货币/触发器/情绪/公共性/实用价值/故事性，6 维度量化传播潜力，5-6分强推，3-4分可做，1-2分慎做
- **信息增量判断**：「大多数人以为____，但实际上____」——填不进去就是信息搬运，不做
- **窗口期分类**：爆发型(72h) / 发酵型(1-2周) / 周期型(提前布局) / 常青型(等触发)
- **冲突设计**：6种冲突类型 × 6维角度切换，穷举最优选题角度
- **情绪映射**：主情绪→次级反应→分享触发→社交货币，设计传播路径
- **Hook 设计**：数字反差/场景还原/时间锚点/直接抛争议/反常识判断，5种开头钩子
- **爆款概率速查**：4维度8分制快速评估

**核心公式**：
> 爆款选题 = 流量潜力 × 评论区讨论价值

## 🔍 深度调研 · 矿工

**文件**：[调研综合手册](researcher-miner/调研综合手册.md)

**核心价值**：从假设构建到多源验证的深度调研全流程协议。不是"搜了就用"，是"扩展→递归→交叉验证→综合"。

**适用场景**：
- 复杂选题的深度调研
- 人物传记的素材挖掘
- 技术/产业的深度拆解
- 事实核查和数据验证
- 竞品分析和行业研究

**关键方法**：
- **扩展法**：一个根问题拆成 15-20 个子问题，从技术/财务/用户/竞品/反面/边缘 6 个维度包围目标
- **递归浏览**：追溯到一手来源（Level 0 搜索→Level 1 文章→Level 2 原始报告→Level 3 数据源→Level 4 采集口径）
- **交叉验证分级**：1源未验证 / 2源初步可信 / 3+源高置信 / 3+源矛盾需列出各方说法
- **声明分级**：A级(已核实) / B级(可信未完全核实) / C级(存疑)，C级必须标注或删除
- **动机分析**：表层动机→行为模式→深层驱动，三层剥洋葱法
- **时间线构建**：骨干节点(转折点) + 支撑事件 + 背景信息，三层结构

**核心原则**：
> 证据质量 > 来源数量。3 个高质量来源 > 20 个互相转载的媒体报道。

## ✍️ 内容写作 · 刀手

**文件**：[写作综合手册](writer-blade/写作综合手册.md)

**核心价值**：基地边缘写作的完整方法论，从开头句法工程到结尾悬念设计。不是"正确但平庸"的内容，是"敢下判断、经得起检验"的内容。793行，最厚的一本。

**适用场景**：
- 所有长文/深度内容写作
- 科技/商业/人物传记类内容
- 视频脚本（含互动式写作指南）
- 新人培训和写作标准制定
- AI 辅助写作后的"去AI味"打磨

**关键方法**：

### 开头句法工程（前3句定生死）
- **介词开头 + 动词延迟**：用"对…来说"开头，主动词推迟到第15字后，制造悬念
- **Hook-Stake-Promise**：第1句钩子、第2句赌注、第3句承诺，三层必须齐全
- **第一句预埋全文主题**：读者读完全文回头看第一句，要有"原来从第一个字就在说这件事"的感觉
- **场景 vs 背景**：从"人"开始，不从"趋势"开始

### 信息密度写作
- **每句话至少干两件事**：叙事+信息、信息+判断、叙事+判断，单一功能的废句一律砍掉
- **括号/插入语 = 信息压缩器**：在不打断叙事节奏的情况下塞入补充事实、反差信息、隐含判断
- **零冗余检测**：逐句尝试删除，删掉后段落依然通顺 = 这句话不该存在
- **长短句交替**：长句承载信息，短句落下判断

### 口语化写作（10大特征）
- 基于锤子/大疆/马斯克/华为/小米/魅族/苹果等 11 篇爆款稿件提炼
- 短句落判断、口语化动词、成语+俚语、对话场景还原、具体数字+极端对比
- 环境描写承载情绪、叙事框架明确化、互动式提问、产品发布会式高潮、历史/流行文化连接
- 每种技巧都有实战案例和使用规则

### 结尾悬念（4种模式）
- 连续"如何"问句（说书式）
- "真正的战争才刚开始"
- 悬念+转折递进
- 说书式+升华

**核心公式**：
> 基地边缘内容 = 高信息密度 × 有判断 × 有锋芒 × 讲人话

## 📝 内容审校 · 质检

**文件**：[质检综合手册](editor-qa/质检综合手册.md)

**核心价值**：系统性审校的完整流程，从宏观结构到表面细节的四轮校对法。不是"通读一遍找错字"，是"分维度、多轮次、有标准"的质量控制体系。

**适用场景**：
- 所有内容发布前的终审
- AI 辅助写作的质量把关
- 事实核查和认知偏差检测
- 内容团队质量标准统一
- Go/No-Go 发布决策

**关键方法**：
- **四轮校对法**：一次只看一件事，每轮只关注当前维度
  - Pass 1：宏观结构（2-3分钟，只看标题+首句）
  - Pass 2：事实准确性（5-10分钟，逐句精读+声明分级）
  - Pass 3：语言表达+AI味（5分钟，大声朗读）
  - Pass 4：表面细节（3分钟，逆向阅读）
- **七类认知偏差检查**：确认偏误/虚假因果/锚定效应/幸存者偏差/可得性偏差/后见之明/权威崇拜
- **AI味评分**：直接性/节奏/信任度/真实性/精炼度，满分50，≥45通过，<35重写
- **Go/No-Go 决策模型**：红线退回(捏造事实/核心数据错误/AI腔严重) / 质量退回(无钩子/无高潮/AI味不达标) / 小改后可发 / 直接可发

**核心原则**：
> 好质检 = 事实准确 × 逻辑自洽 × 去AI味 × 节奏可读

## 🔄 工作流

```
雷达（选题策划）→ 矿工（深度调研）→ 刀手（内容写作）→ 质检（审校终审）→ 发布
```

每个环节有明确的交付标准和退回机制：
- 雷达交付选题必须包含：冲突设计 + STEPPS评分 + 窗口期判断
- 矿工交付调研必须包含：置信等级标注 + 信息空白说明
- 刀手交付稿件必须通过：开头检查 + 信息密度检查 + 风格检查 + 判断检查
- 质检终审必须完成：四轮校对 + AI味评分 + Go/No-Go决策
